# imdb.py
## Python file that returns all IMDB information on a movie as a object.

Usage:
```
from imdb import object
movie=object('Movie Name')
```
### Movie attributes
* title
* year
* rated
* released
* runtime
* genre
* director
* writer
* actor
* language
* country
* awards
* poster
* metascore
* plot
* imdbRating
* dvd
* boxOffice
